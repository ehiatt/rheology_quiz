# resources/

Source material for the quiz — the papers and references behind the questions in
`index.html`. This folder is tracked, but its **contents are local-only**
(gitignored): most of the cited works are paywalled, and this repo is public, so
the PDFs never get committed. The folder itself stays in the tree so the structure
is obvious and the path is stable.

For the full citation list and where each paper lives in my library, see
`../literature.md` (also local-only).

Drop project-specific source files here. Anything that *should* ship publicly
(a CSV of question data, an open figure) can be un-ignored case by case in
`../.gitignore`.
